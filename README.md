# Cpu Monitor Widget Plugin

This project was developed as a cpu monitor widget for windows widgets.

### Internal Settings

```json
{
  "usage_font_size": 18.0,
  "usage_foreground": "#FFBC8F8F",
  "graphic_color": "#347aeb",
  "graphic_timeline": 200.0
}
```
